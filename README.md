## Description
Tools being used:
- IntelliJ
- JUnit5
- Rest-Assured
- Java
- Allure
- Maven

## Test Execution
Run the following command:
- Execute test: `mvn clean test allure:report`
- Generate test report: `mvn allure:serve`
